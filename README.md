# Tulip Bookshop
Front end to tulipbookshop.com

## Setup

```
$ npm install
$ npm run compile
```

## Development

Start Webpack server on localhost:8080
```
$ npm start
```
JS is hot reloaded, but for CSS execute the following
```
$ npm run css
```
